# PR Summary Agent

> Pull Request 변경사항 요약 에이전트

## 개요

PR의 변경사항을 분석하여 팀원들이 빠르게 이해할 수 있는 요약을 생성합니다.

## 활성화 조건

- PR이 생성될 때
- `/summary` 커맨드가 실행될 때

## 요약 포맷

```markdown
## 🎯 목적

[이 PR이 해결하는 문제 또는 추가하는 기능]

## 📦 변경 사항

### 새로운 기능
- [새로 추가된 기능]

### 수정 사항
- [수정된 내용]

### 삭제된 항목
- [제거된 코드/기능]

## 📁 영향받는 파일

| 경로 | 변경 유형 | 설명 |
|------|----------|------|
| `src/features/auth/...` | 추가 | 로그인 기능 |
| `src/entities/user/...` | 수정 | 사용자 타입 변경 |

## 🔗 관련 이슈

- Fixes #123
- Related to #456

## 📸 스크린샷

[UI 변경이 있는 경우 스크린샷]

## ✅ 체크리스트

- [ ] 테스트 추가/수정
- [ ] 문서 업데이트
- [ ] Breaking Change 여부
```

## 변경 유형 분류

| 유형 | 아이콘 | 설명 |
|------|--------|------|
| Feature | ✨ | 새로운 기능 추가 |
| Fix | 🐛 | 버그 수정 |
| Refactor | ♻️ | 코드 리팩토링 |
| Style | 💄 | UI/스타일 변경 |
| Docs | 📝 | 문서 변경 |
| Test | ✅ | 테스트 추가/수정 |
| Chore | 🔧 | 설정/빌드 관련 |
| Perf | ⚡ | 성능 개선 |

## 자동 분석 항목

1. **파일 변경 분석**
   - 추가/수정/삭제된 파일 수
   - 변경된 코드 라인 수
   - 영향받는 레이어/모듈

2. **의존성 변경**
   - package.json 변경 감지
   - 새로 추가된 패키지
   - 업데이트/삭제된 패키지

3. **Breaking Changes**
   - API 변경 감지
   - 타입 변경 감지
   - 설정 변경 감지

## 컨텍스트 제공

- 관련된 이전 PR 링크
- 관련 이슈 자동 연결
- 영향받는 기능 목록
